# ClearVision

Implementation of Wiener deconvolution.

## Overview

This repository contains a very basic Implementation of Wiener deconvolution. Its a script that takes a blurred image and produces a clearer version. 
It builds a soft Gaussian blur model and then uses Wiener deconvolution to undo that blur as much as possible.

## What the Script Does

* Takes a blurred grayscale image
* Builds a Gaussian kernel using a chosen size and sigma
* Uses Wiener deconvolution to restore sharpness
* Saves the final output as **deblurred.png**

## How to Use

1. Place your blurry image in the repo as **blur.png**
2. Run the script
3. Look for **deblurred.png** in the same folder
